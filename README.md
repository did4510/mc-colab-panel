# 🚀 MC Colab Panel

Run a Minecraft server with a professional web panel in Google Colab.

---

## ✨ Features

- 🎮 Minecraft Paper Server
- 💻 Web Dashboard Panel
- 📊 RAM + TPS Graphs
- 🧠 Monaco Editor (VS Code in browser)
- 📂 File Manager
- 🔌 Plugin Installer (Modrinth)
- 📥 Drag & Drop Upload
- 👥 Player Tracking
- 🎨 Glass UI + Dark/Light Mode
- 🌈 Colored Console (ANSI)

---

## 🚀 Run in Google Colab

1. Open `colab/run.ipynb`
2. Run all cells
3. Wait for ngrok link
4. Open panel in browser

---
